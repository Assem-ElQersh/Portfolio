### Colors & Theme
Edit CSS custom properties in `styles/main.css`:
```css
:root {
    --primary-color: #3b82f6;
    --secondary-color: #8b5cf6;
    --accent-color: #06b6d4;
```

### Projects
Update project data in `scripts/projects.js`:
